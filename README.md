# MyDatabase

База данных для системы проката велосипедов. Содержит SQL-скрипты и диаграммы.

## Таблицы

- **City** - город
- **Model** - модель велосипеда 
- **Station** - станция проката
- **Bike** - велосипед
- **Client** - клиент
- **Rent** - аренда
- **Department** - отдел
- **Employee** - сотрудник
